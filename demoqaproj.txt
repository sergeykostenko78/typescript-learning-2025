TECHNICAL REQUIREMENTS:

1. PAGE OBJECT MODEL:
   - Create BasePage (if not exists - reuse from Week 5)
   - Create 5+ specialized pages:
     * FormsPage
     * AlertsPage
     * WidgetsPage
     * TablesPage
     * HomePage
   - Each page must have methods, not just locators
   - Use inheritance from BasePage

2. CUSTOM FIXTURES (MANDATORY - you must create these yourself):
   - Create authFixture.ts with pre-loaded page fixture
   - Create apiMockFixture.ts for API interception setup
   - Create visualTestFixture.ts with screenshot comparison utilities
   - All fixtures must follow pattern:
```typescript
     export const test = base.extend<YourFixtureType>({
         yourFixtureName: async ({ page }, use) => {
             // setup
             await use(result);
             // teardown
         }
     });
```

3. API MOCKING:
   - Use page.route() to intercept API calls
   - Mock successful responses
   - Mock error responses (500, 404)
   - Verify UI handles both scenarios

4. VISUAL TESTING:
   - Install: npm install -D @playwright/test
   - Use page.screenshot() for baseline
   - Use expect(page).toHaveScreenshot() for comparison
   - Store screenshots in tests/visual-regression/

5. ADVANCED INTERACTIONS:
   - Frame switching: page.frame() or page.frameLocator()
   - Alert handling: page.on('dialog', dialog => dialog.accept())
   - Date picker: click calendar, select date programmatically
   - Drag and drop: locator.dragTo()
   - Wait for dynamic elements: waitForSelector with timeout

6. DATA MANIPULATION:
   - Use .map() to transform data
   - Use .filter() to find specific items
   - Use .sort() to verify sorting
   - Use .reduce() if needed for calculations

7. TYPESCRIPT INTERFACES:
   - FormData interface for form inputs
   - TableRow interface for table data
   - ApiResponse interface for mocked API data
   - MockConfig interface for API mock settings

API MOCKING EXAMPLE STRUCTURE:
```typescript
// In apiMockFixture.ts or directly in test
await page.route('**/api/users', route => {
    route.fulfill({
        status: 200,
        contentType: 'application/json',
        body: JSON.stringify({ users: [...] })
    });
});
```

VISUAL TESTING EXAMPLE:
```typescript
await page.goto('https://demoqa.com');
await expect(page).toHaveScreenshot('homepage.png', {
    maxDiffPixels: 100
});
```

ALERTS HANDLING EXAMPLE:
```typescript
page.on('dialog', async dialog => {
    expect(dialog.message()).toContain('expected text');
    await dialog.accept('optional input');
});
await page.click('button-that-triggers-alert');
```

FRAMES HANDLING EXAMPLE:
```typescript
const frameLocator = page.frameLocator('#frame-id');
await frameLocator.locator('button').click();
```
